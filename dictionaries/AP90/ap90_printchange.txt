This file documents changes made to the digitization of AP90 in which the
digitization is intentionally different than the printed edition.
The reasons for these changes are various.
This file was begun 11-25-2015, so there may be some changes prior to this
date which should be documented here but have not thus far been so 
documented.

In conversion to modern IAST, numerous (3000+) changes were made. 
Surely, many of these changes could be considered as print changes.
However, due to the large number of changes, it is not practical to 
individually examine the printed text to determine which should be
classified as print changes.  The file 'temp_allchanges.txt' mentioned
in the above reference identifies most of the changes made.
